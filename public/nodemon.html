Nodemon

Nodemon at NPM

Nodemon helps you develop node projects by continously restarting your server so you do not have to do it whenever you make changes!

Installing nodemon is easy, simply pull up your favourite terminal or powershell and write the following line

npm install -g nodemon

This uses NPM to install nodemon globally (-g) on your machine

Instead of using the node app.js command to run your program you can now type the followig

On Windows:

nodemon.cmd app.js

Mac:

nodemon app.js
